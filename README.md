# 📷 pystagram

## 🛠 주요 기능

- 사용자 회원가입 / 로그인
- 게시물 CRUD (이미지 업로드 포함)
- 좋아요 / 댓글 기능
- 팔로우 / 팔로워
- 템플릿 렌더링 및 static 파일 처리

## 📁 프로젝트 구조

| 폴더 / 파일 | 설명 |
|-------------|------|
| `config/` | Django 프로젝트 설정 |
| `posts/` | 게시글 관련 모델, 뷰, API |
| `users/` | 사용자 관련 기능 (회원가입, 로그인 등) |
| `templates/` | HTML 템플릿 |
| `static/` | 정적 파일 (CSS, 이미지 등) |
| `media/` | 업로드된 사용자 이미지 |
